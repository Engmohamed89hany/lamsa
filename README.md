# 🍽️ Lamsa - Food Ordering Website

Lamsa is a modern and responsive food ordering website with a clean UI and smooth user experience.

## 🚀 Live Demo

https://lamsa3.netlify.app/

## 🛠️ Built With

- React
- Tailwind CSS
- JavaScript
- React Icons

## 📌 Features

- Responsive Design
- Modern UI
- Food Categories
- Smooth Navigation
- Fast Performance

## 🎯 Project Goal

Built to practice frontend development and improve skills in building responsive modern web applications.
